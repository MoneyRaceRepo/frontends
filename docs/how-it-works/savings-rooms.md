# Savings Rooms

## What are Savings Rooms?

Savings Rooms are the core unit of Money Race. Each room is a group savings challenge where participants commit to weekly deposits and compete to stay consistent.

---

## Room Types

### Sprint Rooms (4 weeks)

Short-term challenges for quick wins:
- Great for beginners
- Lower commitment
- Fast rewards cycle

### Standard Rooms (8 weeks)

The most popular format:
- Balanced duration
- Good for building habits
- Optimal reward accumulation

### Marathon Rooms (12 weeks)

For serious savers:
- Maximum savings potential
- Larger reward pools
- Highest achievement badges

---

## Room Parameters

| Parameter | Range | Description |
|-----------|-------|-------------|
| **Duration** | 4-12 weeks | Challenge length |
| **Weekly Target** | $10-$500 | Minimum weekly deposit |
| **Max Participants** | 5-50 | Room capacity |
| **Penalty Rate** | 5-20% | Fee for missed deposits |
| **Strategy** | 3 options | Investment approach |

---

## Creating a Room

1. Navigate to Dashboard → "Create Room"
2. Set your parameters:

```
Room Name: [Your Room Name]
Duration: [4/8/12 weeks]
Weekly Target: [$___] USDC
Max Participants: [___]
Penalty Rate: [___]%
Strategy: [Conservative/Balanced/Aggressive]
```

3. Make your initial deposit
4. Share the room link to invite others

---

## Joining a Room

Requirements:
- Connected account (wallet or zkLogin)
- Sufficient USDC for initial deposit
- Room has available spots
- Room hasn't started yet

---

## Room Lifecycle

```
1. OPEN     → Room created, accepting participants
2. STARTING → Minimum participants reached
3. ACTIVE   → Challenge in progress
4. ENDING   → Final week, last deposits
5. COMPLETE → Winners determined, rewards distributed
6. CLOSED   → All funds withdrawn
```
